# Bryan Stanski's Style Stage

Features and design choices used on my page:

- including a reset/normalize library
- using custom properties
- using fluid sizing and spacing
- typography effects
- styling the background(s)
- the layout at larger widths
- styling the links
- styling the lists
- styling the block quote
